## How to Create a Similar Project?

**Step-1**  
Initialize the project with `npm`:

```bash
npm init
```
**Step-2**
Install Typescript with `npm`:
```bash
npm install -D typescript
```
**Step-3**
Install Express with `npm`:
```bash
npm i express
```
